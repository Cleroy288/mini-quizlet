# Intello

A simple Quizlet-like quiz app built for an interview in ~1h30.

## Tech Stack

- C# / .NET 10
- [.NET MAUI](https://learn.microsoft.com/en-us/dotnet/maui/what-is-maui?view=net-maui-10.0) (Multi-platform App UI)
- MVVM architecture

## Run

```bash
dotnet build -t:Run -f net10.0-maccatalyst
```

Requires .NET 10 SDK installed.
